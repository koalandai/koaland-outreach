# Koaland AI — Outreach (Intelligence OS)

Outbound cockpit for prospect discovery, auditing, email generation, and follow-ups.

- `index.html` — front-end dashboard (works standalone with localStorage persistence)
- `backend/` — TypeScript serverless backend (Vercel + KV) for production use

## Run frontend locally
```bash
npx serve -p 8883 .
```
Open: http://localhost:8883

## Deploy backend
See `backend/README.md` for Vercel deployment instructions.
