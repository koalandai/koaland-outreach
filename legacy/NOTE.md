# Legacy standalone dashboard

These files (`index.html`, `style.css`, `koaland-logo.png`, dated June 21) are the original single-file outreach dashboard, built before the acquisition engine and the cockpit under `backend/frontend/`.

They are kept for reference only. Nothing in the build, the Vercel deploy, or `backend/local-server.js` uses them. The live app is `backend/` (run it with `cd backend && npm install && node local-server.js`).

Safe to delete if you no longer need the old design as a reference.
